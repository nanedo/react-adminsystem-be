# 接口文档格式

```js
{
  "status": 0,
  "msg": "",
  "data": {}
}
```

每个接口必定包含这三项，其中
 - status默认的值有 0（成功）、 1（失败）、 10（未登录）；部分接口会有自己的值，以该接口为准
 - msg包含接口错误信息等
 - data包含返回数据

 > 接口地址： http://localhost:8083/
